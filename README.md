# ☕ Café Expresso

## 📌 Descrição do Projeto

O Café Expresso é um sistema de autoatendimento para cafeterias, onde os clientes podem realizar pedidos em um totem digital sem enfrentar filas.

## 🎯 Objetivos

* Reduzir filas
* Agilizar o atendimento
* Evitar erros nos pedidos

## 🧩 Funcionalidades

* Cadastro de produtos
* Adição de itens ao pedido
* Cálculo automático do valor total
* Controle de status do pedido

## 🏗️ Estrutura

* Produto
* ItemPedido
* Pedido
